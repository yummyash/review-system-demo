# Review System with Local Storage

A responsive review system that stores submissions in browser's local storage and displays them in real-time.

## Features
- User-friendly review submission form
- 5-star interactive rating system
- Persistent storage using Local Storage
- Responsive design for all devices
- Review statistics and analytics
- Clear all reviews functionality

## How to Use
1. Fill in your username
2. Write your review comment
3. Select a rating (1-5 stars)
4. Click "Submit Review"
5. See your review appear in the comments section

## Technical Details
- Built with vanilla HTML, CSS, and JavaScript
- Uses localStorage for data persistence
- No external dependencies required
- Mobile-responsive design
